Instalação:

1) Extraia o ficheiro para um diretório.

2) Abra o terminal no diretório onde o programa está presente.

3) $gcc -Wall *.c -o main

4) $./main

5) Dentro do menu principal, escolha a opção "1" para fazer a simulação do supermercado.

6) Dentro do menu principal, escolha a opção "2" para uma informação mais detalhada do projeto elaborado.

7) Dentro do menu principal, escolha a opção "3" para sair do programa.
